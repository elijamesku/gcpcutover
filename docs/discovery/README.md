# Discovery & Assessment Artifacts

These templates are **Phase 1 deliverables** for a GCP data migration. Fill them in during discovery; they feed planning, wave ordering, and risk mitigation.

| Artifact | Purpose |
|----------|---------|
| [Migration inventory](migration-inventory.md) | What systems and data exist; size, type, priority. |
| [Dependency map](dependency-map.md) | How systems depend on each other (app → DB → storage). |
| [Data classification](data-classification.md) | Sensitivity, retention, residency, and compliance. |
| [Risk list](risk-list.md) | Risks and mitigations for the migration. |

Update these as the project evolves. They are living documents until cutover.
